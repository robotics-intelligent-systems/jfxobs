# Open Banking System

CashMan H.A - Financial Banking System. Bancolombia Scaffolding of Clean Architecture. Kill Bill  OpenSaaS billing. FinGPT is an open-source financial large language models. An orchestration-based saga implementation reference in a serverless architecture. Servicebot is an XaaS (Anything-as-a-service) platform. invoice2data is a command line tool and Python library that automates the extraction of key information from invoices to support your accounting process. Factur-X is a Franco-German e-invoicing standard which complies with the European e-invoicing standard EN 16931. PyFinTS is a pure-python implementation of FinTS (formerly known as HBCI), a online-banking protocol commonly supported by German banks. The Open Bank Project is an open source RESTful API platform for banks that supports Open Banking, XS2A, PSD2 and Open Finance through access to accounts, transactions, counterparties, payments, entitlements and metadata - plus a host of internal banking and management APIs. Apache Fineract is open source software for financial services, designed to create a cloud-ready core banking system that enables digital financial services for everyone, including the unbanked and underbanked. Waltz is an open-source Enterprise Architecture platform that enables organizations to visualize and define their technology landscape. FDC3 is an open standard for applications on financial desktop to interoperate and exchange data with each other. Fluxnova is a flexible framework for workflow and process automation. Hibiscus is a free online banking application for Linux. FinRobot in an open-source AI agent platform for financial analysis using LLMs. PIXIU project is designed to support the development, fine-tuning, and evaluation of Large Language Models (LLMs) in the financial domain. Machine learning for General Ledger provide a complete Data Science framework for Finance Data Analytics. Bigcapital is financial accounting with intelligent reporting

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
